# ygtwa

A Go-based, self-hosted WhatsApp Web REST API and MCP bridge.

## Highlights

- Multi-device oriented setup
- REST API layer for messaging workflows
- Webhook forwarding support
- Chat history access for automation scenarios
- Built for self-hosted deployments

## Status

Early public release.

## Repository Goal

This repository is intended to package practical WhatsApp automation capabilities into a Go service that can be deployed and controlled in real environments.
